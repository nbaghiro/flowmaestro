# Marketing Site Screenshots

This directory contains product screenshots used throughout the FlowMaestro marketing site.

## Screenshot Naming Convention

All screenshots now require both light and dark mode versions:

- `{name}-light.png` - Light mode screenshot
- `{name}-dark.png` - Dark mode screenshot

The site will automatically display the correct version based on the user's theme preference.

## Image Guidelines

1. **Format**: PNG preferred for crisp UI screenshots
2. **Quality**: Capture at 2x resolution if possible for retina displays
3. **Privacy**: Blur/hide any real user data, emails, or sensitive info
4. **Consistency**: Use consistent zoom levels across screenshots
5. **Theme Matching**: Ensure light screenshots are taken in light mode, dark in dark mode
6. **Clean data**: Ensure workflows/agents have professional-looking names
