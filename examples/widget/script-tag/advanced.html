FlowMaestro Widget - Advanced Example

Demonstrates all configuration options and programmatic control

Setup Required: Replace YOUR_SLUG in the script tag with your actual chat interface slug before testing.

Configuration Options

The widget supports these data attributes:

Attribute Values Default Description
data-position bottom-right, bottom-left bottom-right Position of the widget bubble
data-initial collapsed, expanded collapsed Initial state when page loads

Bottom Right (Default)

<script src=".../widget/slug.js" async></script>

Bottom Left

<script src=".../widget/slug.js" data-position="bottom-left" async></script>

Start Expanded

<script src=".../widget/slug.js" data-initial="expanded" async></script>

All Options

<script src=".../widget/slug.js" data-position="bottom-left" data-initial="expanded" async></script>

Programmatic Control

Control the widget via JavaScript using the global FlowMaestroWidget object:

Note: After destroying, you'll need to reload the page to restore the widget.

JavaScript API

Method Description
FlowMaestroWidget.open() Opens the chat window
FlowMaestroWidget.close() Closes the chat window
FlowMaestroWidget.toggle() Toggles the chat window open/closed
FlowMaestroWidget.destroy() Removes the widget from the DOM entirely
