FlowMaestro Widget - Basic Example

This is the simplest possible integration of the FlowMaestro chat widget. Look at the bottom-right corner of your screen to see the chat bubble.

Setup Required: Replace YOUR_SLUG in the script tag below with your actual chat interface slug.

How It Works

The widget is loaded with a single script tag. It automatically:

  • Fetches the chat interface configuration
  • Renders a floating bubble button
  • Opens an iframe chat window when clicked

The Code

<script
  src="https://static.flowmaestro.ai/widget/customer-support-agent-chat.js"
  data-api-url="https://api.flowmaestro.ai"
  async
></script>

Try It

Click the chat bubble in the bottom-right corner to open the chat interface.
