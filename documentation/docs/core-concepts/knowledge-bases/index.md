# Knowledge Bases

Give your agents access to custom knowledge.

## What is a Knowledge Base?

A knowledge base is a collection of documents that gets:

- **Processed** — Text extracted and chunked
- **Embedded** — Converted to vector representations
- **Indexed** — Stored for fast semantic search

## Use Cases

- **Customer Support** — FAQ docs, product manuals, policies
- **Sales** — Product specs, pricing, case studies
- **Internal** — Company wiki, procedures, guidelines

## Key Features

| Feature                | Description                                         |
| ---------------------- | --------------------------------------------------- |
| **Multiple Formats**   | PDF, Word, text, markdown, and more                 |
| **URL Import**         | Import content from web pages                       |
| **Integration Import** | Import from Google Drive, Dropbox, Notion, and more |
| **Semantic Search**    | Find content by meaning, not just keywords          |
| **Auto-Sync**          | Keep content up to date automatically               |
| **Continuous Sync**    | Automatically sync changes from connected apps      |

## Document Sources

FlowMaestro supports three ways to add documents:

### 1. File Upload

Upload files directly from your computer. Supports PDF, Word, text, markdown, HTML, JSON, and CSV.

### 2. URL Import

Scrape content from web pages. Optionally follow links to import multiple pages.

### 3. Integration Import

Import documents from connected apps with optional continuous sync:

- **Google Drive** — Import files and folders
- **Dropbox** — Import files and folders
- **Notion** — Import pages as markdown
- **Confluence** — Import pages as markdown
- **Box** — Import files and folders
- **OneDrive** — Import files and folders

Integration imports support:

- **Folder sync** — Keep an entire folder in sync
- **File selection** — Pick specific files to import
- **Search import** — Import files matching a search query
- **Scheduled sync** — Automatically check for updates (configurable interval)
