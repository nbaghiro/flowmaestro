# Knowledge Bases

Give your agents access to custom knowledge.

## What is a Knowledge Base?

A knowledge base is a collection of documents that gets:

- **Processed** — Text extracted and chunked
- **Embedded** — Converted to vector representations
- **Indexed** — Stored for fast semantic search

## Use Cases

- **Customer Support** — FAQ docs, product manuals, policies
- **Sales** — Product specs, pricing, case studies
- **Internal** — Company wiki, procedures, guidelines

## Key Features

| Feature              | Description                                |
| -------------------- | ------------------------------------------ |
| **Multiple Formats** | PDF, Word, text, markdown, and more        |
| **URL Import**       | Import content from web pages              |
| **Semantic Search**  | Find content by meaning, not just keywords |
| **Auto-Sync**        | Keep content up to date automatically      |
